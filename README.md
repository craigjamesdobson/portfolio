# Craig Dobson Portfolio

A minimalist Astro portfolio based on the visual direction of the Void theme, rebuilt as a focused online CV.

## Stack

- Astro 5
- Tailwind CSS 4 (via `@tailwindcss/vite`)
- TypeScript strict mode
- pnpm

## Commands

```bash
pnpm install
pnpm dev
pnpm check
pnpm build
pnpm preview
```

## Environment

- `SITE_URL` - production URL used for canonical and OG metadata

## Content source

Core profile content is structured in `src/data/cv.ts`.
